# Quotes Listing Application

A simple, clean React application built with Vite to display a collection of inspiring quotes.

## Features
- View a curated list of quotes.
- Fast and responsive UI.
- Built with React and Vite.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

## Technologies Used
- React
- Vite
- CSS
